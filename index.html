ABC News Debate Chatbot

A chatbot "trained" on the transcript of the ABC News debate between Kamala Harris and Donald Trump. "trained" means that the chatbot is "aware" of the transcript and can answer questions about it.

About This Project

This project was built with the following tools:

  • yt-dlp to download the YouTube debate
  • ffmpeg to convert the video to mp3
  • AssemblyAI to transcribe the audio with speaker detection
  • DifyAI to create the chatbot
  • HTML hosted on GitHub Pages
